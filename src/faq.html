Why this website?

There a lot of materials online on performance optimization. However, they are pretty much scatter all over place, and since it deals with a lot of details, the presentation could be a bit boring. This website hopefully provides a good and fun starting points for beginner.

How accurate are the benchmark results?

I tried to be accurate, and all the codes can be found in Github so you can check and validate the results yourself. However, even if the microbenchmark is correct, reading conclusion from that needs to be careful:

  • Microbenchmark measure only a small code snippet. The actual performance gain/loss could be
  • Differences in hardware, compiler, OS, etc could result in wildly different benchmarking result.

The example code is not meant to provide an exact judgement of how code should be written. Instead, it only serve as example on certain concepts
